MySQL+3307端口新增

发表于 2019-02-20 | 更新于: 2019-02-21 |
字数统计: 484 | 阅读时长 ≈ 1
  • 本教程是在Windows下新建一个mysql的3307端口，在新端口下操作数据库不会影响到原来3306端口的数据

本地环境

本地安装的是MySQL5.7，采用安装版安装，不是解压版

一，找到MySQL安装目录

  • 在mysql服务中查询安装目录参数

  • 可以查询到basedir的参数，也就是本地的安装目录，进入目录可以看到如下结构

二、复制安装文件并配置

    • 先将这三个复制到D:\soft\MySQL Server 5.7下的3307文件夹下，（3307文件夹需要自己新建）
